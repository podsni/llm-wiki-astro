# Peluncuran Claude 3

## Ringkasan

[Anthropic](/sources/anthropic) merilis keluarga **Claude 3** pada 4 Maret 2024 — model multimodal (vision + text) dalam 3 tier.

## Varian

- **Claude 3 Haiku** — tercepat, paling murah
- **Claude 3 Sonnet** — balanced
- **Claude 3 Opus** — paling kuat

## Pencapaian

- **Opus SOTA** di:
  - MMLU 86,8%
  - GPQA 50,4%
  - GSM8K 95% (matematika)
  - HumanEval 84,9% (coding)
- **200K context window**
- **Vision** — SOTA di banyak benchmark vision
- **Multilingual** — kuat di banyak bahasa

## Dampak

- **Anthropic** menjadi frontier model resmi
- **Claude 3 Opus** klaim mengungguli GPT-4 (saat itu)
- **Vision** — memicu era multimodal untuk semua frontier model
- **Mendorong** Claude 3.5 Sonnet (Juni 2024) — mengungguli GPT-4o
