# Pokémon TCG Rilis di Jepang

## Ringkasan

**Pokémon Trading Card Game** dirilis di **Jepang** pada **20 Oktober 1996** oleh **MediaFactory** dan **Creatures Inc.**, dengan **Nintendo** sebagai publisher.

## Sejarah

- **1990** — Satoshi Tajiri konsep Pokémon (Game Boy)
- **1996-02** — Pokémon Red & Green (Game Boy) — ledakan
- **1996-10-20** — Pokémon TCG di Jepang
- **1998** — Ditampilkan di World Hobby Fair
- **1998-12** — Versi Inggris dari Wizards of the Coast
- **1999-01** — Base Set di AS
- **2003** — The Pokémon Company didirikan
- **2006+** — Expansi global
- **2016+** — Pokémon TCG Online
- **2019+** — Pokémon TCG Live

## Set Pertama (Jepang)

- **拡張パック 第1弾** (1996-10-20) — 102 kartu
- **拡張パック 第2弾** (1996-12-15) — Jungle equivalent
- **拡張パック 第3弾** (1997-04-01) — Fossil equivalent
- **拡張パック 第4弾** (1997-06-21) — Base Set 2 equivalent

## Game

- **60-kartu deck**
- **6 jenis tipe energi**: Grass, Fire, Water, Lightning, Psychic, Fighting
- **Evolusi**: Basic → Stage 1 → Stage 2
- **Trainer**: Item, Supporter, Stadium
- **Format**: Standard, Expanded, Legacy (rotasi)

## Distribusi

- Jepang
- AS (1999-)
- Eropa (1999-)
- Global (2000+)
- 16+ miliar kartu dicetak sampai 2024

## Pemimpin

- **Tsunekazu Ishihara** — President The Pokémon Company
- **Akio Dobo** — kepala designer kartu
- **Mitsuhiro Arita** — illustrator legendaris
