# Peluncuran Claude 3.5 Sonnet

## Ringkasan

[Anthropic](/sources/anthropic) merilis **Claude 3.5 Sonnet** pada 20 Juni 2024 — model frontier yang **mendominasi** di coding, reasoning, dan vision. Pada peluncurannya, Claude 3.5 Sonnet mengungguli GPT-4o, Gemini 1.5 Pro, dan Llama 3 70B di banyak benchmark.

## Pencapaian

- **Coding**: SWE-bench 49% (vs GPT-4o 38%), HumanEval 93%
- **Reasoning**: MMLU 88,7%, GPQA 59,4%
- **Vision**: multimodal, SOTA di banyak vision benchmark
- **Speed**: 2x lebih cepat dari Claude 3 Opus
- **Cost**: 1/5 harga Opus

## Fitur Baru

- **Artifacts** — window interaktif untuk kode, dokumen, visual
- **200K context window**
- **Vision** — SOTA di dokumen, chart, diagram

## Dampak

- **Anthropic** jadi pemimpin coding & vision
- **Cursor, Codeium, GitHub Copilot** — Claude 3.5 Sonnet jadi backend
- **Valuation** Anthropic naik $60B+
- **Standard** untuk AI assistant enterprise

Claude 3.5 Sonnet dianggap **'GPT-4 moment'** untuk Anthropic — model frontier yang di-acknowledge industry.
