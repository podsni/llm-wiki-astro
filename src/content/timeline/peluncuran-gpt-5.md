# Peluncuran GPT-5

## Ringkasan

[OpenAI](/sources/openai) merilis **GPT-5** pada 7 Agustus 2025 — frontier model yang **mengintegrasikan** kemampuan **reasoning** (seperti o1, o3) ke dalam model default. Tonggak era agentic AI mainstream.

## Fitur

- **Reasoning built-in** — tidak perlu pilih model berbeda
- **Multimodal** — teks, gambar, audio, video
- **Agentic** — dapat menggunakan tool, browse web, run code
- **Long context** — 1M+ token (rumor)
- **~500B parameters** (rumor)

## Dampak

- **ChatGPT 5** (consumer) — agent default
- **Microsoft Copilot** di-upgrade ke GPT-5
- **OpenAI API** dengan reasoning control
- **Operator** — agentic browser automation

## Era Pasca

Bersama Claude 4.5 Sonnet, Gemini 2.5, dan DeepSeek-V4, GPT-5 adalah salah satu dari **4 frontier model** yang mendorong **era agentic AI 2025-2026**. Tidak ada satu model pun yang mendominasi — masing-masing punya kekuatan.
