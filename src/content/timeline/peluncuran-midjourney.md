# Peluncuran Midjourney

## Ringkasan

**Midjourney** adalah model text-to-image proprietary yang beroperasi lewat **Discord** — pendekatan unik. Dikenal karena output yang artistik dan berkualitas tinggi.

## Sejarah

- **Februari 2022** — v1 (closed beta)
- **Juli 2022** — v2, open beta
- **November 2022** — v4 — kualitas lompatan besar
- **Maret 2023** — v5 — mendekati realistik
- **Desember 2023** — v6 — lebih artistik
- **2024-2025** — v7 dengan video generation

## Pendekatan

- **Subscription-only** — tidak ada API
- **Discord-based** — semua interaksi lewat Discord server
- **Komunitas** — lebih dari 16 juta anggota Discord
- **Harga** — $10-$120/bulan

## Pencapaian

- **v5** (2023) — memenangkan banyak benchmark "kualitas"
- **v6** (2023) — mampu render teks dalam gambar
- **v7** (2025) — video generation

## Pesaing

- **DALL-E 3** (OpenAI) — ChatGPT integration
- **Stable Diffusion XL** (open-source)
- **Adobe Firefly** — integrasi Creative Cloud
- **Flux** (open-source, Black Forest Labs)
- **Imagen 4** (Google)
