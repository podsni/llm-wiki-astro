# Yu-Gi-Oh! TCG Rilis

## Ringkasan

**Yu-Gi-Oh! Trading Card Game** resmi dirilis di **Jepang** pada **4 Februari 1999** oleh **Konami**.

## Sejarah

- **1996** — Manga Yu-Gi-Oh! dimulai (Shonen Jump)
- **1998** — Yu-Gi-Oh! Toei anime (limited)
- **1998-04** — Yu-Gi-Oh! Game Boy
- **1999-02-04** — Yu-Gi-Oh! TCG (Konami, Jepang)
- **2000-04** — Yu-Gi-Oh! Duel Monsters anime
- **2002** — Yu-Gi-Oh! The Movie
- **2008** — Yu-Gi-Oh! 5D's
- **2011** — Yu-Gi-Oh! Zexal
- **2015** — Yu-Gi-Oh! ARC-V
- **2017** — Yu-Gi-Oh! VRAINS
- **2020** — Yu-Gi-Oh! SEVENS
- **2023** — Yu-Gi-Oh! GO RUSH
- **2022-07-06** — Kazuki Takahashi meninggal (tenggelam di Okinawa)

## Set Pertama (Jepang)

- **Volume 1** (1999-02-04) — Yu-Gi-Oh! Starter Box
- **Volume 2** (1999-04-01) — Starter Box
- **Booster 1** (1999-05-01)

## Game

- **40-60 kartu deck**
- **Main Deck, Extra Deck, Side Deck**
- **Summon**: Normal, Tribute, Ritual, Fusion, Synchro, Xyz, Pendulum, Link
- **Speed meta** — banyak deck selesai dalam 1-2 turn
- **Format**: OCG (Asia), TCG (selainan Asia)

## Kartu Ikonik

- **Exodia the Forbidden One** — instant win
- **Blue-Eyes White Dragon** — Seto Kaiba
- **Dark Magician** — Yugi
- **Black Luster Soldier** — kartu termahal ($120K+)

## Pasar

- $200-300M+ revenue
- YCS (Yu-Gi-Oh! Championship Series) — turnamen
- Duel Links (digital, 2017-)
- Master Duel (digital, 2022-)
