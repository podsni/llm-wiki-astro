# Peluncuran Claude 4.5 Sonnet

## Ringkasan

[Anthropic](/sources/anthropic) merilis **Claude 4.5 Sonnet** pada 15 November 2025 — model frontier yang **mendominasi** di:
- **Coding** (SWE-bench, HumanEval, LiveCodeBench)
- **Agentic workflows** (Claude Code, Computer Use)
- **Reasoning** (terutama dengan extended thinking mode)
- **Tool use** (parallel, multi-step)

## Kemampuan

- **Context window**: 1M+ token (extended)
- **Extended thinking** — mode reasoning dalam
- **Computer Use** generasi ke-3 — kontrol browser & OS
- **Agentic coding** — Claude Code dominan
- **Long horizon tasks** — multi-hour execution

## Pendorong Agentic AI

Bersama GPT-5, o1/o3, dan Gemini 2.5 Thinking, Claude 4.5 Sonnet menjadi **salah satu dari 4 frontier model** yang mendorong **era agentic AI 2025-2026**:

- **Claude Code** — agentic coding di terminal
- **GitHub Copilot** (Claude sebagai backend) — coding agent
- **Cursor Composer** (Claude backend) — IDE
- **Anthropic API** — untuk perusahaan
- **AWS Bedrock** — distribusi enterprise

## Anthropic 2025-2026

- **Valuation**: $60B+ (awal 2025), $200B+ (akhir 2025)
- **Pendapatan**: $1B ARR (awal 2025), $5B+ ARR (akhir 2025)
- **Pemimpin** di segment enterprise (banking, healthcare, government)
- **Safety** — Constitutional AI, mechanistic interpretability
