# Ravensburger

## Profil

**Ravensburger** adalah perusahaan mainan dan board game asal Jerman, didirikan 1883 di Ravensburg, Jerman. Sekarang salah satu publisher board game dan puzzle terbesar di Eropa.

## Sejarah

- **1883** — Didirikan oleh Otto Maier
- **1900s** — Puzzles, board games
- **1950s** — Merek Ravensburger
- **1980s** — Ekspansi global
- **2023** — Disney Lorcana TCG

## Produk

### Trading Card Game
- **Disney Lorcana** (2023-) — TCG dengan karakter Disney

### Board Game
- **Scotland Yard** (1983) — klasik
- **Ramses Pyramid** (2000s)
- **Pictopia** (trivia)
- **Labyrinth** (1986)
- **Enchanted Forest** (1982)

### Puzzle
- **1000-piece puzzles**
- **3D puzzles**
- **Disney puzzles**
- **National Geographic puzzles**

## Pasar

- Jerman, Austria, Swiss (kantor pusat)
- Amerika, Eropa
- €600M+ revenue (2023)
- Disney Lorcana: $200M+ revenue (2023-2024)
