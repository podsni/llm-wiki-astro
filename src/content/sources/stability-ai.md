# Stability AI

## Profil

**Stability AI** adalah perusahaan AI asal **London, Inggris** yang fokus pada **generative AI open-source**. Paling dikenal sebagai sponsor utama **Stable Diffusion**.

## Sejarah

- **2019** — didirikan oleh [Emad Mostaque](https://twitter.com/EMostaque)
- **2022** — Stable Diffusion 1.4 (Agustus) — viral
- **2022** — pendanaan $101M, valuation $1B
- **2023** — Stable Diffusion XL, valuation $4B (peak)
- **2024** — Emad Mostaque mundur dari CEO (November 2023), digantikan Shan Shan Wong & Christian Laforte
- **2024** — Stable Diffusion 3, 3.5
- **2025** — fokus enterprise + API

## Produk

- **Stable Diffusion** (1.4, 2, XL, 3, 3.5, Turbo)
- **Stable Video Diffusion** (video generation)
- **Stable Audio** (music generation)
- **Stable LM** (LLM open-source)
- **Membership API** (commercial)

## Komunitas

- **Civitai** — model & LoRA marketplace
- **Hugging Face** — model repository
- **AUTOMATIC1111** — WebUI
- **ComfyUI** — node-based workflow
- **LAION** — dataset sponsor (5B image-text pairs)

## Pendanaan

- **2022** — Coatue, Lightspeed
- **Peak valuation** — $4B
- **2024** — pendanaan lebih kecil, pivot ke enterprise
