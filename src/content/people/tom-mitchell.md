# Tom Mitchell

## Biodata

- **Lahir**: 1951
- **Profesi**: Machine learning researcher

## Kontribusi

- **Machine Learning textbook** (1997) — standar industri selama 20+ tahun
- **CMU Machine Learning Department** (2006) — departemen ML pertama di dunia
- **NELL** (Never-Ending Language Learner) — web-scale ML yang belajar terus
- **TD-Learning** (earlier work, bersama)

## Definisi ML

Definisi formal Mitchell (1997):
> *"A computer program is said to learn from experience E with respect to some class of tasks T and performance measure P, if its performance at tasks in T, as measured by P, improves with experience E."*

## Karier

- **Rutgers University** (1978-1994)
- **Carnegie Mellon University** (1994-)
- **Founding chair of CMU ML Department** (2006)
- **National Academy of Engineering**

## Legacy

Mitchell adalah **pionir ML akademis** — buku textbook-nya melatih **generasi ML researcher**, dan departemen ML CMU menjadi model untuk program ML di seluruh dunia.
