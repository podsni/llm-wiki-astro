# Arthur Samuel

## Biodata

- **Lahir**: 5 Desember 1901, Emporia, Kansas
- **Meninggal**: 29 Juli 1990
- **Profesi**: Pioneer in computer gaming and machine learning

## Kontribusi

- **Checkers-playing program** (1952) — salah satu program game pertama
- **Istilah 'machine learning'** (1952) — pertama kali dicetuskan
- **Alpha-beta pruning** (1950-an) — algoritma search tree
- **Rote learning** dan **generalization** dalam AI
- **IBM** (1949–1966)

## Karier

- **Bell Labs** (1928)
- **University of Illinois** (1946)
- **IBM** (1949–1966) — Poughkeepsie Lab

## Pencapaian

- Program checkers-nya **mengalahkan pemain Connecticut state champion** (1962)
- Menulis paper seminal *"Some Studies in Machine Learning Using the Game of Checkers"* (1959)
