# Claude Code

## Definisi

**Claude Code** adalah **agentic coding tool** dari [Anthropic](/sources/anthropic) yang berjalan di **terminal**. Memimpin di agentic coding 2025-2026.

## Fitur

- **Repository-aware** — memahami seluruh codebase
- **Multi-file editing** — dapat mengedit banyak file sekaligus
- **Sub-agents** — mendelegasikan task ke sub-agent
- **MCP integration** — Model Context Protocol
- **Bash tool** — menjalankan command
- **Web fetch** — mengambil data
- **Plan mode** — membuat plan sebelum eksekusi

## Workflow

1. User menulis prompt bahasa natural
2. Claude Code menganalisis codebase
3. Membuat plan
4. Eksekusi multi-langkah dengan tool
5. Self-review & test
6. Hasil: patch, PR, atau fix

## Era Pasca

Bersama **Cursor**, **Codeium**, **GitHub Copilot Coding Agent**, **Codex CLI**, dan **Gemini CLI**, Claude Code adalah salah satu agentic coding tools paling banyak digunakan 2025-2026.
