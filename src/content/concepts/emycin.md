# EMYCIN

## Definisi

**EMYCIN** (Essential MYCIN) adalah shell expert system generik yang dihasilkan dari [MYCIN](/concepts/mycin) pada 1980 oleh Edward Feigenbaum & Bruce Buchanan.

## Fitur

- **Backward chaining** inference engine
- **Certainty factors** (handle uncertainty)
- **Domain-independent** — bisa dipakai untuk expert system apapun
- **Rule-based** (IF-THEN)

## Signifikansi

EMYCIN memungkinkan pembangunan expert system **tanpa membangun inference engine dari awal** — cukup isi knowledge base. Ini memicu ledakan expert system 1980-an:

- **PUFF** (pulmonary function)
- **SACON** (structural engineering)
- **DIPMETER** (geology)
- **CLOT** (blood clotting)
- **DRILLING ADVISOR** (oil drilling)

EMYCIN adalah **langkah pertama kommoditisasi AI** — template untuk commercialize AI.
