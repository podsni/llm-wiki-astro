# Gemini CLI

## Definisi

**Gemini CLI** adalah agentic coding tool dari [Google DeepMind](/sources/google-deepmind) yang berjalan di terminal. Diperkenalkan 2025.

## Fitur

- **1M context window** (Gemini 2.5 Pro)
- **Multimodal** — bisa baca gambar di terminal
- **Function calling** — tool use
- **Search grounding** — Google Search
- **Code execution** — sandbox

## Pesaing

- **Claude Code** (Anthropic)
- **Codex CLI** (OpenAI)
- **Qwen Code** (Alibaba)
- **DeepSeek CLI**

## Era 2025-2026

Gemini CLI adalah salah satu dari banyak agentic coding tools 2025-2026. Backed by Google's massive AI infrastructure.
