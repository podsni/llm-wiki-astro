# Machine Learning

## Definisi

**Machine learning** (ML) adalah sub-bidang [kecerdasan buatan](/concepts/kecerdasan-buatan) yang memberikan komputer kemampuan untuk **belajar dari data** tanpa diprogram secara eksplisit.

Istilah ini dicetuskan oleh [Arthur Samuel](/people/arthur-samuel) pada 1952 saat membangun program [checkers](/concepts/checkers) yang belajar dari bermain melawan dirinya sendiri.

## Paradigma

1. **[Supervised learning](/concepts/supervised-learning)** — belajar dari data berlabel (X, Y)
2. **[Unsupervised learning](/concepts/unsupervised-learning)** — belajar dari data tanpa label
3. **[Self-supervised learning](/concepts/self-supervised-learning)** — pre-training dengan masked prediction
4. **[Reinforcement learning](/concepts/reinforcement-learning)** — belajar dari reward
5. **Few-shot learning** — belajar dari beberapa contoh saat inference

## Algoritma Klasik

- **Linear regression** (1800-an, statistik)
- **Decision trees** (ID3, CART — 1980-an)
- **Support Vector Machines** ([Vapnik](/people/vladimir-vapnik), 1995)
- **k-Nearest Neighbors**
- **Naive Bayes**
- **Random Forests** (2001)
- **Gradient Boosting** (XGBoost, LightGBM)

## Era Modern

Sejak 2012, [deep learning](/concepts/deep-learning) mendominasi ML di computer vision, NLP, dan speech recognition.
