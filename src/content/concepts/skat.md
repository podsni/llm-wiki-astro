# Skat

## Definisi

**Skat** adalah permainan kartu **trick-taking** 3 pemain (atau 4) dari Jerman, menggunakan deck Jerman 32 kartu. Game nasional Jerman.

## Sejarah

- **1817-1835** — Berschanskat (Prusia)
- **1840s** — Skatsspiel berkembang
- **1886** — Skatordnung (rules) distandarisasi
- **Sekarang** — game nasional Jerman

## Aturan

- **3 pemain** (1 declarer, 2 defender)
- **Deck 32 kartu** (7, 8, 9, 10, J, Q, K, A per suit)
- **Suit**: Acorn, Bell, Heart, Leaf
- **Bidding** — declarer menentukan game
- **Multiplier** — Clubs < Spades < Hearts < Diamonds < Grand

## Game Type

- **Suit game** — clubs, spades, hearts, diamonds
- **Grand** — hanya Jacks (Unter, Ober) trump
- **Null** — tidak ada tricks dimenangkan (zero game)
- **Hand** (Schneider), **Schwarz** (zero opponent)
- **Ouvert** — kartu terbuka
- **Ramsch** — semua kalah-kan

## Skat Tournament

- **Deutsche Skatliga**
- **International Skat Players Association**
- **World Championship**
