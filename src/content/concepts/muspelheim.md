# Muspelheim

## Definisi

**Muspelheim** (*Múspellsheimr*, "Rumah Muspel") adalah salah satu dari **9 dunia** Norse, primordial dunia **api**. Rumah **Surt** dan rakyat api Muspel. Saat Ragnarök, Surt akan membakar dunia.

## Karakteristik

- **Api** — primordial
- **Panas** — ekstrem
- **Cahaya** — menyilaukan
- **Asal** — di awal waktu

## Penduduk

### Surt

- **Pemimpin** Muspelheim
- **Raksasa api** (fire giant)
- **Pedang** — lebih terang dari matahari
- **Ragnarök** — bakar 9 dunia

### Sinmara

- **Istri** Surt
- **Bangsawan Muspel**

### Anak-Anak Muspel

- **Muspel** — ratus atau lebih
- **Api** — terkait dengan Surt

## Kisah Penciptaan

- **Sebelum Ymir** — Muspelheim dan Niflheim sudah ada
- **Percikan api** dari Muspelheim + es Niflheim = **raksasa Ymir** (?)
- Atau: Api + es = tetesan air = hidup
- **Ymir** jadi primoridal creature
- **Sapi Audumbla** menjilat es
- **Búri** keluar (leluhur dewa)

## Ragnarök

- **Surt** bakar dunia
- Pedangnya menyala
- Api menelan segalanya
- 9 dunia hancur
- **Awal baru** muncul dari lautan

## Etimologi

- **Muspel** — mungkin "pencuri" atau "api"
- **Muspell** — dalam Puisi Völuspá sebagai "muspel" api
