# Turing Test

## Definisi

**Turing Test** (juga **imitation game**) diusulkan oleh [Alan Turing](/people/alan-turing) dalam makalah *"Computing Machinery and Intelligence"* (1950) di jurnal *Mind*.

## Prosedur

1. Ada tiga pihak: manusia (A), mesin (B), dan interrogator (C)
2. C mengetik pertanyaan ke A dan B lewat terminal
3. C tidak tahu siapa manusia, siapa mesin
4. Jika C tidak bisa membedakan dengan akurasi > 50% dalam 5 menit, mesin **lolos**

## Kritik

- **Chinese Room** (John Searle, 1980) — simulasi pemahaman ≠ pemahaman
- **Total Turing Test** — termasuk persepsi fisik
- **Lovelace Test** — kreativitas mesin
- **Winograd Schema Challenge** — common sense

## Status Modern

LLM seperti GPT-4, Claude, Gemini sering **mengelabui manusia** dalam percakapan singkat, tapi belum ada yang "resmi" lolos tes standar.
