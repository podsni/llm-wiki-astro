# Surt

## Definisi

**Surt** (*Surtr*, "Yang Hitam" — atau "Yang Berasap") adalah raksasa **api** dari **Muspelheim** dalam mitologi Norse. Memimpin rakyat Muspel.

## Ciri Khas

- Api — primordial
- Pedang menyala — lebih terang dari matahari
- Raksasa — besar
- Kekuatan — api

## Keluarga

- Istri: Sinmara (?)
- Tinggal: Muspelheim

## Peran

- Menjaga Muspelheim
- Bakar 9 dunia saat Ragnarök
- Bertarung dengan Freyr

## Ragnarök

- Datang dari Selatan — dengan api
- Pedang — menyala-nyala
- Api membakar semua
- Asgard — terbakar
- Midgard — terbakar
- Yggdrasil — terbakar
