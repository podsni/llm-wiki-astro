# Agentic AI

## Definisi

**Agentic AI** adalah sistem AI yang dapat **bertindak secara otonom** — merencanakan, menggunakan tool, mengeksekusi aksi, dan menyesuaikan strategi berdasarkan feedback.

## Arsitektur

Komponen kunci:
- **LLM** sebagai "otak" (planning, reasoning)
- **Tool use** (search, code execution, API, browser, file system)
- **Memory** (short-term context + long-term storage)
- **Planning** (decompose goal → steps)
- **Reflection** (evaluate, retry, adjust)

## Era 2024-2026

- **Claude Computer Use** (Anthropic, Oktober 2024)
- **Devin** (Cognition Labs, 2024)
- **OpenAI Operator** (Januari 2025)
- **GitHub Copilot Coding Agent** (2025)
- **Claude Code** (2025)
- **Codex CLI** (2025)
- **Gemini CLI** (2025)
- **Agentic IDEs** (Cursor Composer, Windsurf)
