# OpenAI Operator

## Definisi

**Operator** adalah AI agent dari [OpenAI](/sources/openai) yang dapat **mengontrol browser web** dan melakukan tugas multi-langkah secara otonom. Diluncurkan **Januari 2025**.

## Fitur

- **Browser control** — Chrome/Chromium
- **Visual understanding** — lihat halaman
- **Multi-step tasks** — plan + execute
- **Form filling** — otomatis
- **Shopping** — cari, bandingkan, beli
- **Research** — browse banyak halaman
- **Email** — drafting, sending

## Pesaing

- **Claude Computer Use** (Anthropic, Oktober 2024) — pelopor
- **Manus AI** (Ciklum, 2025)
- **Google Project Mariner**
- **Open-source** — Skyvern, Browser-Use

## Backend

- **Operator 1** — GPT-4o dengan vision
- **Operator 2** (2025) — o1/GPT-5 dengan reasoning
