# Codex CLI

## Definisi

**Codex CLI** adalah agentic coding tool dari [OpenAI](/sources/openai) yang berjalan di terminal. Memungkinkan developer untuk melakukan coding task multi-langkah dari terminal.

## Fitur

- **GPT-5 backend** — model frontier
- **Sandbox execution** — menjalankan code dengan aman
- **Multi-file editing**
- **Tool use** — bash, file system, dll
- **Plan + execute**

## Era

- 2021 — Codex API (Codex model)
- 2022-2023 — GitHub Copilot (Codex model)
- 2024-2025 — Codex Agent
- 2025 — Codex CLI (terminal)
- 2026 — ChatGPT Codex (integrated)
