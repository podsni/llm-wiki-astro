# Unsupervised Learning

## Definisi

Model belajar dari data tanpa label — menemukan pola, cluster, atau representasi yang berguna.

## Aplikasi

- **Clustering** — K-means, DBSCAN, hierarchical
- **Dimensionality reduction** — PCA, t-SNE, UMAP
- **Anomaly detection**
- **Density estimation**
- **Generative modeling** (GAN, VAE)
