# Cursor Composer

## Definisi

**Cursor Composer** adalah mode agentic di IDE **Cursor** (fork VS Code). Memungkinkan AI agent untuk melakukan editing multi-file dengan codebase context.

## Pendiri Cursor

- **Aman Sanger** (CEO)
- **Michael Truell**
- **Sualeh Asif**
- **Arvid Lunnemark**

## Fitur

- **Multi-file editing** — agent mengedit banyak file sekaligus
- **Codebase-aware** — memahami seluruh repo
- **Agent mode** — eksekusi multi-langkah
- **Chat + inline + agent** modes
- **Terminal integration**
- **Git integration**
- **Image input** — screenshot UI

## Backend

- **Claude 4.5 Sonnet** (default, sangat populer)
- **GPT-5** (OpenAI)
- **Cursor custom models** (rumor)

## Era 2024-2026

Cursor valuation $9B+ (akhir 2025). ARR $500M+ (akhir 2025). Pendiri muda (semua 2023 graduates MIT). Pesaing: **Windsurf** (Codeium), **Claude Code** (Anthropic), **GitHub Copilot**.
