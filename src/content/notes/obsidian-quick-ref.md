# Obsidian — quick reference

Catatan ringkas tentang setup Obsidian saya.

## Plug-in penting
- **Dataview** — query note sebagai tabel
- **Templater** — template untuk note baru
- **Calendar** — navigasi by date
- **Graph Analysis** — lihat central nodes
- **Excalidraw** — diagram di dalam note

## Vault structure
- `00-inbox/` — note baru masuk sini
- `10-projects/` — project aktif
- `20-areas/` — area of responsibility
- `30-resources/` — referensi, tidak aktif
- `40-archive/` — sudah selesai, diarsip

## Daily note
- Setiap pagi bikin note dengan nama `YYYY-MM-DD`
- Isi: tasks, log, links
- Lihat [[Digital Garden]] untuk ide serupa

## Wikilink pattern
- `[[page-name]]` — link ke note lain
- `[[page-name|alias]]` — link dengan display text
- `[[page-name#heading]]` — link ke section
